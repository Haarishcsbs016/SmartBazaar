📌 Overview

SmartBazaar is a modern e-commerce application designed to provide a seamless and personalized shopping experience. It integrates AI capabilities to enhance product discovery and user engagement while ensuring secure and efficient transactions.

✨ Key Features
🔐 JWT-based User Authentication & Authorization
🛍️ Product Management (CRUD operations with admin control)
🔍 Smart Search & Advanced Filtering
🛒 Cart & Wishlist Functionality
💳 Secure Payment Integration (Razorpay / Stripe)
📦 Order Management & Tracking
⭐ Reviews & Ratings System
🤖 AI-powered Recommendations & Chatbot
📊 Admin Dashboard for analytics and management
🧠 AI Capabilities
Personalized product recommendations using user behavior
AI chatbot for handling customer queries
Smart search suggestions using LLM APIs
Enhanced user experience through intelligent insights
